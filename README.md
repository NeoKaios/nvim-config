## :fire: Main remaps

`<leader>` = `<Space>`

### Insert

- `kj` to `<Esc>`

### Normal

- `<leader>w` to `:w`
- `<leader>q` to `:q`
- `<leader>f` to Focus Nvim-tree
- `<leader>t` to Toogle Nvim-tree
- `<leader>mp` to Open Markdown Preview
- `<C-h|j|k|l>` to `<C-w>h|j|k|l`
